# TOGETHER

CHLNGERS (챌린져스)를 어플리케이션를 모방한 TOGTHER PROJECT입니다.

TOGETERH는 스스로 돈을 걸고 실천해 나가는 목표 달성 플랫폼 입니다.
혼자 목표를 달성하는게 아닌 같은 목표를 가진 사람들과 함께 목표를 달성합니다. 100% 달성하면 전액 페이백 + 상금이 주어집니다.

1. 지키고 싶은 나와의 약속을 고르세요.
2. 그리고 결심에 돈을 걸어두세요.
3. 실천할때마다 환급! 못지키면 벌금
